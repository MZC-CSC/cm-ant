# Overview
## 클라우드 마이그레이션 전환 상태 검증 프레임워크
복잡한 클라우드 마이그레이션 수행 절차의 자동화 및 이전 후 성능 및 비용 예측이 가능한 워크플로우 통합 관리 및 상태 검증 기술 개발을 목표로 크게 두 가지 기술로 나뉜다.
- 목표 클라우드 인프라 전환 비용 검증 및 예측 기술
- 목표 클라우드 인프라 온디맨드 성능 평가 및 검증 기술
